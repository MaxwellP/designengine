    Instructions:
  1. Open the console by pressing F12 in Chrome.
  2. In order to select a game run the readJSON("") funciton, with either "ttv2" (tic-tac-toe) or "rps" (rock-paper-scissors) inside the quotes.
  3. If you typed in ttv2, use the following instructions.
    • To make a move, run the ticTacToeMove(x,y) function, with any number between 1 and 3 (inclusive) for the values of x and y.
    • The computer will randomly select a move. If the game is not over, continue to enter moves until a winner is declared.
    If you typed in rps, use the following instructions.
    • To make a move, run the rockPaperScissorsMove("") function, with either "Rock", "Paper", "Scissors" inside the quotes.
    • The computer will randomly select a move and the winner will be determined.
  4. If you wish to reset the games or run a different game, run the functions refered to in bullet 2.
